### [notebooks](notebooks)
Contains Python Jupyter notebooks (.ipynb)

Files should be prefixed in the order they should run, ie.
- 00_BAIS3250_project_scraping_APIdata.ipynb

Contains Python scripts (.py)

Files should be prefixed in the order they should run, ie.
- 00_file_1.py
- 01_file_1.py
- 02_file_1.py
